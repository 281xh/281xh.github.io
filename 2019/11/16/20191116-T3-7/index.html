快速排序（递归&非递归）-T3-7

Posted on 2019-11-16 Edited on 2019-11-18 In algorithm practise

Description

实现快速排序。

Input

输入第一行为用例个数， 每个测试用例输入的每一行代表一个数组，其中的值用空格隔开，第一个值表示数组的长度。

Output

输出排序的数组，用空格隔开，末尾不要空格。

Sample

Sample Input 1
1
13 24 3 56 34 3 78 12 29 49 84 51 9 100

Sample Output 1
3 3 9 12 24 29 34 49 51 56 78 84 100

java

总结&思路

八大算法之一，利用栈实现非递归的版本，首先需要掌握常规的递归实现，将递归语句替换成符合运算顺序逻辑的入栈出栈操作即可，需要注意的是入栈出栈顺序。
